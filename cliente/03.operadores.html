Operadores

Los operadores son símbolos reservados para realizar ciertas manipulaciones, cálculos, en general operaciones sobre los datos de un programa.


    let x = 1 + 1; // los unos son operandos y el + es el operador

De la misma forma, tenemos más operadores aritméticos.

  • Operadores Aritméticos

    • Son los operadores para realizar cálculos numéricos: + - * /, etc.

    Operador Descripción Ejemplo
    + Suma a + b
    - Resta a - b
    * Multiplicación a * b
    ** Exponente (ES2016)

    x^y -> x ** y     		x * x -> x ** 2

    2 ** 3 -> 2 * 2 * 2 -> 8
    / División a / b

    b no debe ser 0 -> Infinity
    % Módulo (Resto de divisón)
    5/3 = 1, R = 2     		5 % 3 -> 2

    2 es el resto de la división entera

    4 % 2 -> 0 (división exacta)
    ++ Incremento: aumenta en 1 el valor 
    
    let x = 1;
    x++; 
    // ahora x vale 2
    -- Decremento: disminuye en 1 el valor 
    
    let x = 3;
    x--; 
    // ahora x vale 2

  • Operadores de Asignación

    • Son los operadores que realizan una operación aritmética y además asignan el resultado de dicha operación a la variable. Si bien la notación puede ser más corta, no siempre escribir menos es más claro.

    Operador Ejemplo Equivale a:
    Asignación:

    =     		x = y x = y
    Suma con asignación:

    +=     		x += y 
                    
    let x = 1; // x vale 1 
    x += 2;   // x vale 3
    		x = x + y
    Resta con asignación:

    -=    		 x -= y x = x - y
    Multiplicación con asignación:

    *=    		 x *= y 
    
    let x = 1; // x vale 1
    x *= 2;  // x vale 1*2 -> 2
    x *= 0; // x vale 0
    x *= 100 // x vale 0
    		x = x * y
    División y asignación:

    /=     		x /= y

    nótese que no está definido
    dividir por cero         		x = x / y
    Resto y asignación:

    %=     		x %= y 
    
    let x = 10; // x vale 10
    x = 10 % 3; // x vale 10 / 3 -> 3 resto 1 => 1
    		x = x % y
    Exponenciación y asignación:

    **=     		x **= y 
    
    let x = 2; let y = 3; 2 ** 3 -> 8
    x **= y; // x vale 8
    		x = x ** y
